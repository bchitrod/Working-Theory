# Working Theory

Local portfolio site for Bhakti Chitroda.

## Run locally

From this folder:

```bash
cd /Users/cern/Downloads/Portfolio
python3 -m http.server 8000
```

Then open:

- [http://127.0.0.1:8000](http://127.0.0.1:8000)
